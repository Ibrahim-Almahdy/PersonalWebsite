# Personal Website

A responsive personal portfolio website built using HTML5, CSS3, Bootstrap 5, and Font Awesome. The project is available in both English and Arabic (RTL) versions.

## Live Demo

https://ibrahim-almahdy.github.io/PersonalWebsite/

## Preview

![Personal Website Preview](./img/preview.png)

## Features

* Responsive Design
* English & Arabic (RTL) Support
* Responsive Navigation Bar
* Hero Section
* Bootstrap Grid System
* Bootstrap Cards
* Bootstrap Carousel
* Newsletter Subscription Section
* Portfolio Gallery
* Font Awesome Icons
* Modern Footer Design

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Font Awesome
* Bootstrap Grid System
* Bootstrap Components
* Bootstrap Utilities
